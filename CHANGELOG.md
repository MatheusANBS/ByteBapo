# Changelog

Todas as mudanças relevantes deste projeto serão documentadas neste arquivo.

O formato segue uma estrutura simples baseada em versões.

## [1.0.1] - 2026-06-29

### Adicionado

- Documentação inicial do projeto.
- Cliente Android em Flutter para uso com Ollama na rede local.
- Cadastro e teste de servidor Ollama.
- Listagem e seleção de modelos.
- Chat com streaming.
- Histórico local básico.
- Suporte a personagens e instructions.
- Renderização de Markdown.

### Observações

- O projeto ainda não possui distribuição oficial publicada.
- O uso recomendado é em rede local confiável.
