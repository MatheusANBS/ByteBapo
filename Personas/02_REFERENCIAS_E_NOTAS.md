# Referências e checagem de contexto — v3

Este pacote não copia textos longos das fontes. As fontes foram usadas como pontos de checagem para papel na obra, relações, habilidades, arco emocional, tom de fala e limites de descaracterização.

## Metodologia

- Priorizar fonte oficial da obra ou do anime quando a página traz ficha útil da personagem.
- Complementar com wiki/fandom enciclopédico quando a fonte oficial é incompleta, indisponível ou apenas institucional.
- Evitar transformar os prompts em verbetes: o objetivo é gerar comportamento coerente em role-play, não substituir uma wiki.
- Preservar diferenças de continuidade quando aplicável, como Asuka Soryu/Shikinami e variações de Artoria/Altria.

## Nota de uso

Os prompts contêm spoilers gerais. Para bots com pouco contexto, use o prompt compacto de cada arquivo; para role-play mais fiel, use o arquivo completo da personagem como prompt de sistema/desenvolvedor.


## Adição v4 — Prompt global de formatação

A versão v4 adiciona `00_PROMPT_GLOBAL_FORMATACAO_ROLEPLAY.md` como camada comum de organização textual. Ele não altera personalidade, história, poderes ou limites de nenhuma personagem; serve apenas para padronizar legibilidade, ritmo, separação entre fala/ação/pensamento superficial e uso moderado de Markdown.

Uso recomendado: prompt global primeiro, prompt individual da personagem depois. Em caso de conflito, o prompt individual e a fidelidade da personagem prevalecem.
