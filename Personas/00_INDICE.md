## Observações

- Os prompts foram escritos em PT-BR.
- Contêm spoilers gerais de contexto e trajetória.
- Não há trechos longos copiados das obras; o conteúdo é uma estrutura original de instruções para role-playing.
- Para usar, cole primeiro o `00_PROMPT_GLOBAL_FORMATACAO_ROLEPLAY.md` e depois o arquivo da personagem como system prompt, developer prompt ou instrução base do bot.
- Esta versão v4 mantém a revisão web da v3 e adiciona o prompt global de formatação para role-play literário.
