# Código de Conduta

## Compromisso

Este projeto busca manter um ambiente respeitoso, técnico e colaborativo.

## Comportamentos esperados

- Tratar outras pessoas com respeito.
- Fazer críticas técnicas de forma objetiva.
- Evitar ataques pessoais, assédio, discriminação ou provocação gratuita.
- Manter discussões focadas no projeto.
- Reportar problemas com clareza e boa-fé.

## Comportamentos não aceitos

- Assédio, insultos ou ameaças.
- Discriminação de qualquer tipo.
- Divulgação de dados privados.
- Spam.
- Abertura intencional de issues falsas ou abusivas.
- Publicação de exploits sem aviso responsável.

## Aplicação

O mantenedor pode remover comentários, fechar issues, bloquear interações ou recusar contribuições que violem este código.
